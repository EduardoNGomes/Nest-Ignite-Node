# Forum App

### Neste projeto forum abordados conceitos como DDD, SOLID, Factory Pattern, Adapters, Testes unitarios, Testes E2E

### Tecnologias utilizadas
- TypeScript
- NestJS
- Express
- Multer
- Cloudflare
- Jwt
- Vistest
- Zod
